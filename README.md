# liovr-vcodec
<https://github.com/liovr/liovr-vcodec>

跨平台视频压缩 (编码/解码, 硬件加速) 封装库

[![CI](https://github.com/liovr/liovr-vcodec/actions/workflows/ci.yml/badge.svg)](https://github.com/liovr/liovr-vcodec/actions)


## 编译开发

+ 代码格式化:

  ```
  cargo fmt
  ```

+ 编译:

  ```
  cargo build
  ```

+ 测试:

  ```
  cargo test
  ```

+ 代码文档:

  ```
  cargo doc
  ```
